# Slack-notify

## Install
- pip3 install slack-notify
- python3 setup.py install

## Usage
run `slack-notify --token "TOKEN"`, [click here](https://api.slack.com/docs/oauth-test-tokens) to get your code.

## Python Requirements
- pycurl==7.43.0
- ujson==4.35
- ws4py==0.35.0

## Systema Requirements
- xcowsay
